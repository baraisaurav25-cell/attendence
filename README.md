# 🧬 BioTrack: Smart Attendance System
**Department of Biotechnology & Genetic Engineering | Sylhet Agricultural University**

BioTrack is a high-performance desktop application designed for university teachers to automate student attendance and track attendance percentages in real-time.

## ✨ Features
- **Modern UI:** Sleek dark-themed interface built with `CustomTkinter`.
- **Centralized Database:** All records are saved in a single `attendance_report.xlsx` file.
- **Visual Analytics:** - 🟢 Green indicator for $\ge$ 60% attendance.
  - 🔴 Red indicator for < 60% attendance.
- **Search & Filter:** Instant student search by ID.
- **Role-Based Login:** Secure access for Admin and Teachers.

## 🚀 Installation & Usage
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
